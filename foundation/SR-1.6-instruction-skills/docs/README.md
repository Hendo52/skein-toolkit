# SR-1.6 — Instruction and Skill Governance

## Purpose

The project shall maintain path-scoped instruction packs (`.github/instructions/*.instructions.md`) that constrain editing rules by subsystem and domain-specific skill manifests (`.github/skills/*/SKILL.md`) that encode reproducible workflows. Repo memory accumulates cross-session learning from non-obvious bug resolutions and architectural decisions.
